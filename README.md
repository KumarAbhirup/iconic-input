## Installation
- **Step 1:** Install with npm or yarn
```
$ npm i iconic-input --save

----- OR -----

$ yarn add iconic-input
```
- **Step 2:** Import components from *iconic-input*
```
import * as iconicInput from 'iconic-input'
```
